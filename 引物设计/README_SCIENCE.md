# ======================================================================
# 1. 引物设计的基本热力学原理
# ======================================================================
"""
1.1 熔解温度 (Melting Temperature, Tm)
---------------------------------------
Tm 是 50% 的 DNA 双链解离为单链时的温度。

计算方法:
  - Wallace 规则 (适用于 ≤20bp 的短引物):
    Tm = 2°C × (A+T数量) + 4°C × (G+C数量)

  - 最近邻热力学模型 (Nearest-Neighbor, 更精确):
    Tm = ΔH / (ΔS + R × ln(C/4)) - 273.15
    其中 ΔH 和 ΔS 根据相邻碱基对的堆叠自由能计算。
    这是 Primer3 和 MRPrimerV 使用的方法。

  - 盐校正公式 (SantaLucia, 1998):
    Tm(Na+) = Tm(1M NaCl) + 16.6 × log₁₀([Na+]) - 273.15

我们的参数选择:
  - PCR 引物 Tm: 58-62°C
    → 这是 qPCR 的标准退火温度区间 (通常退火温度 = Tm - 5°C)
    → 确保在 53-57°C 退火时有足够的特异性
  - qPCR 引物 Tm: 58-60°C (更窄的范围)
    → qPCR 退火和延伸通常合并为一步 (60°C)
    → 所有引物必须在同一温度下高效工作

1.2 GC 含量
------------
GC 含量影响:
  - 引物-Template 结合强度 (GC 配对有 3 个氢键, AT 只有 2 个)
  - Tm 值 (GC 越高, Tm 越高)
  - 非特异性结合风险 (GC 过高会导致错误配对)

我们的参数: 40-60%
  → 过低 (<40%): 结合不稳定, 需要更低的退火温度
  → 过高 (>60%): 可能导致非特异性扩增
  → 理想值: 50% (±10%)

1.3 3' 端稳定性 (3' End Stability, ΔG)
---------------------------------------
3' 端是 DNA 聚合酶起始延伸的位置。
3' 端过于稳定 (ΔG 太负) 会导致:
  - 非特异性起始: 即使只有 3' 端匹配, 聚合酶也可能启动延伸
  - 错配延伸: 3' 端的高稳定性掩盖了上游的错配

我们的参数: ΔG ≥ -9 kcal/mol (即不太稳定)
  → 这意味着 3' 端的 5 个碱基不能全是 G/C

3' GC Clamp (3' 端 GC 夹):
  - 3' 端最后一个碱基最好是 G 或 C (更强的结合, 减少"呼吸")
  - 但 3' 端最后 5 个碱基中 GC 不能超过 3 个 (避免过稳定)
  - 这就是 MRPrimerV 的 "3' end stability ≥ -9 kcal/mol" 的含义

1.4 二级结构规避
-----------------
引物自身配对形成二级结构会导致:
  - 引物二聚体 (Primer Dimer): 两条引物互补配对
    → 消耗引物, 降低检测灵敏度
    → 产生非特异性产物
  - 发夹结构 (Hairpin): 单条引物自身回折配对
    → 引物无法与 Template 结合
  - 3' 端杂交: 最危险的, 因为聚合酶会从 3' 端开始延伸

我们的参数:
  - 自身互补: <4bp (严格模式)
  - 引物对互补: <4bp
  - 3' 端互补: <3bp (最严格)

1.5 同聚物 (Poly-X) 规避
-------------------------
连续相同碱基 (如 AAAAAA, GGGGGG) 会导致:
  - 聚合酶滑动 (Slippage): 产生错误长度的产物
  - 非特异性结合: Poly-A 可与 Poly-T 在任何位置结合

我们的参数: 连续相同碱基 ≤5bp
"""

# ======================================================================
# 2. PCR 引物 vs qPCR 引物 vs 简并引物
# ======================================================================
"""
2.1 常规 PCR 引物
------------------
目的: 检测病毒是否存在 (Yes/No 检测)
特点:
  - 较长的扩增子 (250-1000bp)
    → 方便后续 Sanger 测序验证
    → 有足够的序列信息确认病毒身份
  - 较少的引物对 (每物种 3-5 对)
    → 选择最保守区域
  - 可使用凝胶电泳验证

设计策略:
  a) 多序列比对 → 找保守区 (Shannon 熵 < 0.5)
  b) 在保守区内运行 Primer3
  c) 按 Penalty Score 排序

2.2 qPCR 引物 + TaqMan 探针
----------------------------
目的: 实时定量检测病毒载量
特点:
  - 短扩增子 (80-200bp) → 高扩增效率 (>90%)
    → 短扩增子 = 更短的延伸时间 = 更少的非特异性产物
    → 但过短 (<70bp) 会导致引物二聚体与产物的熔解曲线难以区分
  - 需要内部探针 (TaqMan probe)
    → 5' 端荧光基团 (FAM, VIC, HEX 等)
    → 3' 端淬灭基团 (BHQ1, TAMRA 等)
  - 更高的 Tm 要求:
    → 引物: 58-60°C (两步骤 qPCR)
    → 探针: 68-70°C (比引物高 8-10°C, 确保探针先结合)
  - 探针设计规则:
    → 5' 端不能是 G (G 会淬灭荧光信号)
    → 3' 端不能是 G
    → GC 含量 40-60%
    → 长度 18-30bp

    TaqMan 探针工作原理:
    ① 探针在退火/延伸阶段与目标序列特异性结合
    ② Taq DNA 聚合酶的 5'→3' 外切酶活性切割探针
    ③ 荧光基团与淬灭基团分离 → 发出荧光信号
    ④ 每个 PCR 循环荧光信号增强 → 与扩增产物量成正比

  Ct 值与引物质量的关系:
    - 好引物: Ct < 25 (高灵敏度)
    - 可用引脚: 25 < Ct < 30
    - 差引物: Ct > 30 (低灵敏度, 或引物二聚体干扰)

2.3 简并引物 (Degenerate Primers)
-----------------------------------
目的: 检测高度变异的病毒 (同一物种但不同株系差异大)
问题: 如果病毒的高变区在引物结合位点, 单一序列引物可能:
  - 对某些株系完全不能结合 (假阴性)
  - 这会导致检测遗漏关键变异株

解决方案: 使用 IUPAC 简并碱基编码:

  IUPAC 简并码表:
    R = A|G     (purine)
    Y = C|T     (pyrimidine)
    S = G|C     (strong)
    W = A|T     (weak)
    K = G|T     (keto)
    M = A|C     (amino)
    B = C|G|T   (not A)
    D = A|G|T   (not C)
    H = A|C|T   (not G)
    V = A|C|G   (not T)
    N = A|C|G|T (any)

  例如: 如果位点在不同株系中是 A 或 G, 引物中用 R 表示
  → 合成时制造商会在该位点混合 A 和 G 两种碱基

  注意事项:
  - 简并度 (Degeneracy) = 所有可能的引物组合数
    → 例如: "R" 产生 2 种引物, "V" 产生 3 种
    → 总简并度 = 各位置简并度的乘积
  - 简并度过高 (>64) 会导致引物池过于稀释
    → 每条特定引物的浓度太低
    → PCR 效率下降
  - varVAMP 的核心价值:
    → 自动平衡覆盖率和简并度
    → 使用动态规划找到最优简并编码方案
    → 避免不必要的简并引入
"""

# ======================================================================
# 3. 特异性验证的科学基础
# ======================================================================
"""
3.1 BLAST 特异性验证
---------------------
原理: 将引物序列与数据库中的所有序列比对, 检查:
  - 是否只命中目标病毒 (靶标特异性)?
  - 是否命中宿主基因组 (脱靶风险)?
  - 是否命中其他病毒 (交叉反应)?

关键参数:
  - E-value: 期望值, 越低越特异
  - Query coverage: 引物被比对上的长度比例
  - Identity: 匹配碱基比例
  - Mismatch positions: 错配位置 (3' 端错配最致命)

评估标准:
  - 靶标特异性: 所有 top hits 都是目标病毒 → 特异性高
  - 交叉反应: 有非目标病毒的 hits → 需要检查 ΔG
    → 如果脱靶 hit 的 3' 端 ΔG < -9 kcal/mol, 有交叉反应风险
  - 宿主反应: 命中宿主基因 → 可能产生假阳性

3.2 PCR_strainer 热力学分析
----------------------------
PCR_strainer 的核心方法:
  ① TNTBLAST: 快速找到所有与引物有 ≥10bp 匹配的模板区域
  ② ΔG 计算: 使用最近邻模型计算每个引物-模板对的结合自由能
  ③ 3' 端分析: 重点分析引物 3' 端 6bp 的 ΔG
     → 3' 端 ΔG < -9 kcal/mol → 可能发生非特异性起始
     → 这是 PCR_strainer 的独特优势: 不是简单看序列匹配,
       而是计算热力学可行性

  为什么 3' 端更重要?
  - DNA 聚合酶从 3' 端开始添加核苷酸
  - 即使 5' 端有多个错配, 只要 3' 端 6-8bp 完美匹配,
    聚合酶仍然可能启动延伸
  - 这就是 MRPrimerV 严格要求 3' 端稳定性的原因

3.3 引物二聚体
---------------
类型:
  a) 自身二聚体: Fwd-Fwd 或 Rev-Rev
     → 引物自身配对, 无法与 Template 结合
     → 降低有效引物浓度
  b) 交叉二聚体: Fwd-Rev
     → 两条引物互相配对
     → 聚合酶可延伸产生短的非特异性产物
     → 在 qPCR 中会被 SYBR Green 染色 → 假阳性信号
  c) 3' 端交叉二聚体: 最危险
     → Fwd 的 3' 端与 Rev 互补
     → 产生"引物二聚体"产物, 大小约等于两引物长度之和
     → 在 qPCR 中消耗引物和聚合酶 → 降低灵敏度

检测方法:
  将引物和其反向互补序列做滑动窗口比对:
    max_Match(i) = 对于所有可能的起始位置和对齐方式,
                   找到最大连续匹配长度
  如果 3' 端最后 6bp 中有 ≥3bp 互补 → 高风险
"""

# 4. 多重平铺扩增 (Multiplex Tiling) 设计原理
# ======================================================================
"""
Olivar 的设计策略:

4.1 两相 PCR 策略
------------------
Phase 1: 长片段扩增 (1.2-2.5kb)
  - 使用少量引物对覆盖整个基因组
  - 扩增子之间有重叠 (overlap > 100bp)
  - 确保没有覆盖缺口

Phase 2: 短片段扩增 (200-500bp)
  - 以 Phase 1 产物为模板
  - 设计更多引物对, 产生更短的扩增子
  - 用于 Illumina 测序文库构建

4.2 GC 偏差校正
----------------
病毒基因组中的极端 GC 区域:
  - 高 GC (>65%): 引物难以结合, 扩增效率低
  - 低 GC (<35%): 引物结合太弱

解决方案:
  - 在 GC 极端区域增加引物长度 (延长到 25-30bp)
  - 或者使用 GC 增强剂 (如 Betaine, DMSO) — 实验层面
  - Olivar 自动在 GC 偏差区域调整引物设计参数

4.3 引物间兼容性
-----------------
多重 PCR 中所有引物同时在同一个反应管中工作:
  - Tm 必须一致 (所有引物在相同温度下退火)
  - 不能有引物间交叉杂交
  - 扩增子大小需要不同 (电泳区分)

Olivar 通过以下方式确保兼容性:
  ① 全局 Tm 优化 → 所有引物 Tm 接近
  ② 引物对间互补检查 → 避免交叉杂交
  ③ 大小分档 → 不同扩增子大小可区分
"""

# ======================================================================
# 5. 参数选择的科学依据
# ======================================================================
"""
5.1 为什么 Tm 设为 58-62°C?
---------------------------
- Mg²⁺ 浓度 1.5-2.5mM 时的标准 PCR 条件
- 退火温度 = Tm - 5°C ≈ 53-57°C
- 这个温度区间 Taq 聚合酶活性 70-80%
- 过高 → 特异性高但灵敏度低
- 过低 → 灵敏度高但特异性低 (假阳性)

5.2 为什么产物大小设为 100-500bp (常规 PCR)?
---------------------------------------------
- <100bp: 与引物二聚体大小接近, 难以区分
- 100-500bp: 最适合 Taq 聚合酶 (延伸速率 ~1kb/min)
  → 延伸时间只需 30-60秒
- >1000bp: 需要更长延伸时间, 增加非特异性扩增风险

5.3 为什么 qPCR 产物设为 80-200bp?
-----------------------------------
- qPCR 的效率公式: E = 10^(-1/slope) - 1
- 理想效率: 90-110%
- 短扩增子 (<200bp) 几乎总是 100% 效率
- 扩增子长度与 Ct 值的关系:
    Ct_200bp ≈ Ct_100bp + 1 (每个额外 100bp 约延迟 1 个循环)
- 取 80-200bp:
    → 足够长以便与引物二聚体区分 (primer dimer ~40bp)
    → 足够短以保证高效率
    → TaqMan 探针始终在两条引物之间

5.4 为什么引物二聚体检查设为 <4bp?
------------------------------------
- 在 PCR 退火温度 (53-57°C) 下:
  → 4bp 的互补配对 ΔG ≈ -4 to -6 kcal/mol
  → 此 ΔG 太弱, 不足以稳定形成二聚体
- 5bp 或更多: ΔG 足够形成稳定二聚体
- 3' 端的互补更危险 (设为 <3bp):
  → 即使 3bp 的 3' 端互补, 聚合酶也可能延伸

5.5 评分系统的设计
-------------------
引物综合评分 (0-100) = 基准分 100 - 扣分项:

  GC 偏差扣分:
    |GC - 50| × 1.5  → 每偏离 1% 扣 1.5 分

  二聚体扣分:
    3' 交叉二聚体 ≥3bp  → 扣 30 分 (致命)
    引物间交叉二聚体 ≥5bp → 扣 15 分
    自身二聚体 ≥5bp       → 扣 10 分

  Poly-X 扣分:
    连续 5 个相同碱基 → 扣 15 分

  3' GC Clamp 不足 → 扣 10 分

  推荐等级:
    ≥80: RECOMMENDED  — 可直接使用
    60-79: USABLE      — 需要湿实验验证
    40-59: CAUTION     — 仅在无更好选择时使用
    <40: NOT_RECOMMENDED — 不推荐
"""

# 6. 完整的实验验证流程
# ======================================================================
"""
引物设计完成后, 推荐的湿实验验证步骤:

1. 引物合成
   - 常规引物: 脱盐纯化即可
   - qPCR 探针: HPLC 纯化, 确保荧光标记纯度
   - 简并引物: 混合合成 (Hand-mix 或 Machine-mix)

2. 梯度 PCR 优化退火温度
   - 范围: Tm 计算值 ± 5°C
   - 8 个温度梯度: 50, 52, 54, 56, 58, 60, 62, 64°C
   - 选择: 最亮的单一条带所对应的最高温度

3. 灵敏度测试
   - 10倍梯度稀释模板 (10⁶ → 10⁰ copies/μL)
   - qPCR: 确定检测下限 (LOD)
   - LOD 通常在 1-100 copies/μL

4. 特异性测试
   - 阳性对照: 目标病毒 cDNA/RNA
   - 阴性对照: 健康植物 RNA、近缘病毒
   - 无模板对照 (NTC): 水 → 检查引物二聚体

5. 临床样本验证 (如有)
   - 已知阳性样本
   - 已知阴性样本
   - 计算灵敏度、特异性、PPV、NPV
"""
