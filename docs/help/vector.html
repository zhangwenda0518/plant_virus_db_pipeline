Vector-Host DB

Insect vector and plant host interaction network for plant viruses

Vector-Host DB — Vector-Host Database

Overview

Vector-Host DB integrates insect vector and plant host association data for 1,806 viruses, unifying records from the WUR Plant Virus Transmission Database and EPPO Global Database through a dual-source integration strategy. Each association includes key parameters such as vector taxonomy, transmission mode, acquisition period, and inoculation period.

Quick Start

  1. Open /vector/ and enter a virus name in the search box
  2. View the vector list for that virus, including vector species, taxonomic information, and transmission parameters
  3. Filter by vector type: aphid, whitefly, thrips, leafhopper, mealybug, etc.
  4. View transmission modes: persistent, non-persistent, semi-persistent
  5. Browse the host plant range for that virus to understand its natural host spectrum
