Virus Explorer

Interactive multi-panel virus data exploration with 7 linked visualization panels

Virus Explorer — Interactive Dashboard

Overview

Virus Explorer is an interactive viral data exploration tool built with Plotly Dash. It reads data from the Reference DB backend and provides 7 linked panels with multi-dimensional filtering by species, family/genus, and time range. All panel filters update synchronously in real time.

Quick Start

  1. Open /explorer/ and enter a virus species name (e.g. tomato) in the top search box or select a family/genus
  2. Timeline panel: View collection and publication year distributions for selected viruses; drag the time slider to filter by date range
  3. World Map panel: View geographic distribution of virus sequences at country/region level
  4. Genome Viewer panel: Interactive genome length distribution chart, color-coded by family
  5. Taxonomy Tree panel: Hierarchical taxonomy tree showing family/genus/species relationships
  6. Table panel: Full sequence list with Accession, species, length, country, date fields; supports sorting and export
  7. Statistics and Sequence Alignment panels provide statistical summaries and alignment views

Reference: 7-Panel Details

Tip: All panels are linked. For example, clicking a family in the Taxonomy Tree synchronously filters Timeline, Map, and Table to show only that family's sequences. Selecting a year range in Timeline updates all other panels correspondingly.
