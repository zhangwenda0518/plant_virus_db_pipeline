Overview

Symptom Gallery collects 2,757 plant virus disease symptom photos sourced from the EPPO Global Database. Each photo is annotated with host plant, tissue type, symptom type, and photo position, supporting multi-dimensional filtering and browsing. The photo gallery directly links to corresponding virus entries in the Reference DB, enabling a seamless workflow from symptom observation to virus identification.

Quick Start

  1. Open /photos/; the default view displays all photo thumbnails
  2. Use the top host plant filter to select crops of interest (e.g. tomato, rice, citrus)
  3. Use the tissue filter: leaf, stem, fruit, flower, root
  4. Use the symptom type filter: mosaic, chlorosis, necrosis, stunting, ringspot
  5. Click a photo to view the full-size image and detailed information (host, pathogen, symptom description)
  6. Click the virus link below the photo to jump to Reference DB and view genomic information for that virus

Reference: Filter Dimensions

Host Plant

Filter by host plant species. Covers major crops (rice, corn, wheat, potato, tomato, citrus) as well as cash crops and ornamental plants.

Tissue Type

Tissue types where symptoms appear, including leaves (the most common site of symptom manifestation), stems, fruits, flowers, and roots.

Symptom Type

Symptom type classification: mosaic (yellow-green mottling), chlorosis (uniform yellowing), necrosis (tissue death), stunting (growth inhibition), ringspot (concentric ring patterns).

Tip: Combining host and symptom dimensions can quickly narrow the diagnostic scope. When observing similar symptoms in the field, first filter by host, then compare symptom types; after finding matching photos, click the virus link to obtain primers for PCR confirmation.
