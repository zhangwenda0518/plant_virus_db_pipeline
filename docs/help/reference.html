Overview

Reference DB is the core reference sequence database of this platform, integrating all plant virus-related sequences from NCBI GenBank and cross-validated against the authoritative ICTV VMR MSL41 taxonomy. After the seven-step QC pipeline (Host Classify, ICTV Validate, Segmented Detect, Taxonomy Dedup, SeqKit rmdup, MMseqs2 0.98 clustering, vclust Cluster), it produces 199K non-redundant sequences covering 6,182 plant virus species, including viruses, viroids, satellite nucleic acids, and unclassified plant-associated sequences.

Dashboard Panel

The Reference DB homepage provides six visualization dashboards:

  • Sequence Overview: Sequence count distribution by family, for a quick overview of each viral family's scale.
  • Genome Type Distribution: Proportion statistics for ssRNA / ssDNA / dsRNA / dsDNA genome types.
  • Sequence Length Distribution: Histogram of sequence lengths across the database, reflecting different genome size spectra.
  • Host Classification: Virus sequence associations by host plant species/genus, revealing host specificity patterns.
  • Temporal Distribution: Timeline trend of sequence submission years, tracking discovery and research activity changes across viral families.
  • Segmented / Non-Segmented: Paginated browsing of segmented and non-segmented viral genomes with family/genus hierarchical filtering.

Core Features

  • Sequence Search: Supports exact or fuzzy search by virus name, species, family/genus, or GenBank Accession number.
  • Segmented Browser: Designed for multipartite viruses, displaying independent sequences and annotations for each segment.
  • Submit: Users can submit new plant virus sequences; the system automatically performs host classification and ICTV validation before ingestion.
  • Download: Download FASTA sequence files and corresponding metadata tables by family, genus, or segment dimensions.
  • Genome Visualization: View genome structure annotation diagrams for selected virus entries, including CDS and mature peptide cleavage site annotations.
Quick Start
  1. Go to Reference DB Dashboard to view sequence overview and genome type distribution, building an intuitive understanding of the database's scale and composition.
  2. Switch to the Segmented tab to browse multipartite virus segment details by family and genus hierarchy.
  3. Use the top search box to enter a target virus species name or GenBank Accession to precisely locate entries of interest.
  4. On the Download page, select family/genus/segment dimensions to download corresponding FASTA files and metadata tables for downstream analysis.
Reference

Data Source

  • Sequence Data: NCBI GenBank nucleotide database; all plant virus-related entries retrieved programmatically via Entrez.
  • Taxonomy System: ICTV Virus Metadata Resource (VMR) MSL41, providing officially recognized virus species and their higher-order classification.
  • Host Information: Host field from NCBI records processed via NLP and mapped to standardized plant species names.

Download Interface

The download page is organized into three levels:

  • Family Level: Download all sequences by viral family; suitable for large-scale comparative genomics analysis.
  • Genus level: Download by viral genus, suitable for intra-genus genetic diversity studies.
  • Segment level: Download each segment independently for multipartite viruses, suitable for per-segment analysis.

Each download level includes a metadata table (CSV format) with Accession, species, host, genome type, length, and submission date fields.

Tips
Search Tips: Use the full scientific name of a virus species (e.g. Tomato yellow leaf curl virus) for optimal matching precision; abbreviations (e.g. TYLCV) can also be used for fuzzy search.
Download Optimization: When batch-downloading large numbers of sequences, download at the Family granularity to control file size and network transfer time.
Submit New Sequence: Before using the Submit function, ensure sequences have undergone basic vector removal and quality filtering. The system will automatically perform host classification and ICTV cross-validation, and processing results will be displayed on the page.
Multipartite Viruses: For multipartite viruses such as Geminiviridae and Bromoviridae, please view complete segment information on the Segmented page. Each segment is independently numbered (e.g. RNA1, RNA2, DNA-A, DNA-B), and different segments may encode distinct functional proteins.
