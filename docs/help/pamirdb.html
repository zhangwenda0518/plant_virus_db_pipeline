PV-miRNA (PAmiRDB)

Plant antiviral miRNA target prediction workbench with three-algorithm consensus

PV-miRNA — Prediction Workbench Main Page

Overview

PV-miRNA (formerly PAmiRDB) is a plant antiviral miRNA target prediction workbench. Using a miRanda + psRNATarget + RNAhybrid three-algorithm consensus strategy, it systematically predicts miRNA-virus interactions across 402 plant species and 42 virus species. Results are annotated with three-tier consensus labels and displayed with multi-dimensional visualizations including Sankey flow diagrams, force-directed network graphs, genomic position maps, and heatmaps.

Quick Start

  1. Open /pamirdb/ and click a preset example (e.g. 4 miRNAs to CMV RNA3) for a quick experience
  2. Or select from database: (1) Left miRNA Input Panel: choose species and miRNA; (2) Virus Target Panel: select virus; (3) Click Run Analysis
  3. View results: Dashboard statistics panel (Total Hits, Avg Energy, Avg Identity)
  4. Sankey Diagram (left): miRNA to viral protein flow; scroll to zoom
  5. Network Graph (right): Force-directed interaction network; drag nodes
  6. Table pagination: 20 rows/page; Target Gene column shows hit genes (3a / CP / Intergenic)
  7. Click View for details: Alignment (base pairing), Algorithms (per-algorithm scores), Position Map (genomic position + CDS annotation), miRNA Structure

Reference: Consensus Tier

TierCriteriaMeaning
STRICTAll 3 algorithms agree; strong seed pairing; low MFEHigh confidence; recommend prioritizing for experimental validation
MODERATETwo algorithms agreeModerate confidence; suitable for screening candidate targets
LOOSESingle algorithm supportNeeds additional validation; high false positive rate

Three Algorithms

  • miRanda: Sequence complementarity + free energy scoring; ideal for rapid scanning
  • psRNATarget: Plant-specific prediction with mismatch penalty mechanism
  • RNAhybrid: Minimum free energy (MFE) hybridization prediction

Visualization

  • Sankey Diagram: miRNA to viral protein flow; node size proportional to interaction count; supports scroll-wheel zoom
  • Network Graph: Force-directed interaction network; miRNA nodes in green, viral protein nodes in red; supports drag and hover highlighting
  • Position Map: Linear genome map with CDS gene annotations and miRNA binding site positions
Tip: Filter by MODERATE and above first, then export the filtered results. The Position Map requires CDS annotation data for the virus; choose full-length reference genome viruses (e.g. CMV RNA3) to see complete gene tracks.
