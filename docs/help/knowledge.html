Overview

Knowledge RAG is a Retrieval-Augmented Generation (RAG) based plant virology knowledge Q&A system. It indexes the full text of two authoritative databases: Descriptions of Plant Viruses (DPV) and viroidDB. Ask questions in natural language (Chinese or English), get accurate answers with cited sources. Ideal for quickly looking up virus biology, transmission mechanisms, and taxonomic information.

Quick Start

  1. Open /knowledge/ and type your question in the input box (Chinese or English)
  2. Wait for AI to search DPV and viroidDB knowledge bases and generate an answer (typically 5-15 seconds)
  3. Check the cited sources below the answer to verify which DPV entry or viroidDB record the information comes from
  4. Follow-up questions are supported; the system retains context

Data Source

  • DPV (Descriptions of Plant Viruses): Authoritative plant virus description database maintained by the Association of Applied Biologists, covering virus biology, transmission, host range, geographic distribution, and genome organization
  • viroidDB: Comprehensive viroid database containing sequence, structure, host, and pathogenicity information
Tip: Be as specific as possible. Generic queries like "tell me about plant viruses" return overviews, while specific questions like "How is Tomato spotted wilt virus transmitted by thrips in a circulative-propagative manner?" receive much more precise answers. No need to repeat context when following up.
