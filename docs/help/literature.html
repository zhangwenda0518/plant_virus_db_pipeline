Automated PubMed plant virus literature monitoring with AI-generated summaries

Overview

Literature Tracker automatically monitors PubMed for plant virology literature with daily updates of newly published papers and weekly retrospection of the latest publications. Currently tracking 6,575 papers, each with AI-generated summaries covering research methods and key findings. Papers are classified into seven research directions with full-text search and multi-dimensional filtering.

Quick Start

  1. Open /literature/ — the default view shows the latest paper list
  2. Use the left-side category filter to select: Detection Methods, Evolution & Diversity, Host Resistance, Biotechnology, Epidemiology, Viroid, Virus-Host Interaction
  3. Click a paper title to expand the abstract and read the AI-generated summary
  4. Click the PubMed link to access the full text
  5. Use the search box to search by keyword (virus name, method, author)
