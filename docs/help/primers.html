Overview

Primer Database contains 57,072 plant virus detection primer pairs covering 4,789 virus species. Primer types include PCR, qPCR, degenerate primers, and whole-genome tiling amplicons. Every primer pair has been BLAST-validated and annotated with specificity and reliability scores.

Primer Types

  • PCR Primers: Standard PCR amplification primer pairs for routine virus detection and identification.
  • qPCR Primers: Real-time quantitative PCR primers for viral load quantification and sensitive detection.
  • Degenerate Primers: Primers with degenerate bases, designed for highly variable virus populations to cover more variant strains.
  • Tiled Amplicons: Overlapping primer sets covering the full viral genome, suitable for whole-genome amplification and sequencing.

Three-Tier Scoring System

Each primer pair is comprehensively evaluated based on BLAST validation results, specificity, dimer risk, and GC content:

  • Recommended: High BLAST specificity, no significant dimer risk, Tm and GC% within optimal range. Recommended for priority use.
  • Usable: Minor dimer or mismatch risks present but still functional under most conditions. Can serve as alternatives.
  • Caution: BLAST validation found significant cross-reactivity or secondary structure issues. Consider only when no better options exist.
Quick Start
  1. Enter a target virus name in the Primer Database search box (e.g. Tomato brown rugose fruit virus); the system automatically matches relevant primers.
  2. Browse the returned primer list; each row shows primer name, score tier, product length, and target gene region.
  3. Click any primer entry to enter the detail page and view full information: forward and reverse sequences, Tm, GC%, product length, BLAST validation results, target hit rate, and specificity score.
  4. Use the score filter to display only Recommended grade primers to quickly lock in the best candidates.

API Access

GET /api/all_primers?format=csv returns the complete primer dataset as a CSV file containing all primer pairs and their validation information.

Tips
Search Strategy: Use a virus genus name (e.g. Begomovirus) to retrieve primers for all virus species under that genus at once. Degenerate primers generally offer broader coverage and are suitable for genus-level detection needs.
Primer Selection Principles: Prioritize primer pairs marked as "Recommended". If the target virus has high variability and recommended primers are insufficient, consider degenerate primers, but pay extra attention to Tm matching and dimer risks.
Tiling Primer Sets: Whole-genome tiling primers are suitable for MinION or Illumina whole-genome sequencing workflows. Each set contains dozens of overlapping primer pairs; note the amplification efficiency and segment coverage uniformity of each pair during use.
Batch Export: To integrate large amounts of primer data into local experimental designs, directly obtain the complete dataset via the API endpoint /api/all_primers?format=csv.
