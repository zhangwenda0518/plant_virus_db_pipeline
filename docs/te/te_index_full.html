About This Collection

This database provides a curated reference set of plant transposable elements (TEs) and endogenous viral elements (EVEs). The data is derived from two complementary sources:

U-RVDB v32.0 — The 11.1M-sequence unclustered reference virus database. We extracted the PLN (Plant & Fungal) division (22,210 sequences), classified them using the RVDB keyword-based system into LTR-RETRO, ENRV, ERV, EX, and UNASSIGNED categories, then filtered for Viridiplantae organisms only.

NCBI nuccore — Targeted searches for plant endogenous viral elements using terms: endogenous[Title] OR EPRV[Title] OR EVE[Title] OR florendovirus[Title] OR badnavirus[Title] in Viridiplantae. After removing PREDICTED, patent, mRNA, and UNVERIFIED entries, 597 genuine plant EVE sequences were added.

All sequences are in FASTA format with RVDB-style headers: >acc|SOURCE|ACCESSION|Description|Organism|Division|Date. Clustered at 95% identity with MMseqs2 (--min-seq-id 0.95 -c 0.8 --cov-mode 1).

Processing Pipeline

U-RVDB v32.0 → PLN Division (22,210) → Keyword Classification → Viridiplantae Filter → NCBI Supplement (v2) → MMseqs2 95% Cluster → Quality Filter (no patent/rRNA/yeast) → Final Reference Set

Script: rvdb_extract_plant.py — single-pass scan of 183GB U-RVDB FASTA with inline RVDB classification logic, taxonomy filtering, and MMseqs2 clustering.
