Plant Virus Reference Database

All plant virus sequences from NCBI GenBank, cross-referenced with ICTV VMR MSL41. Covers viruses, viroids, satellites and unclassified plant-associated sequences with host, taxonomy, genome type and collection metadata.

About This Database

Curated plant virus sequences integrating NCBI GenBank records with authoritative ICTV taxonomy. The pipeline performs:

Host Classify→ ICTV Validate→ Segmented Detect→ Taxonomy Dedup→ SeqKit rmdup→ MMseqs2 0.98→ vclust Cluster
