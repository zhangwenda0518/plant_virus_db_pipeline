Plant Virus Reference Database

A comprehensive, ICTV-verified reference database of plant virus genomes. Built from NCBI GenBank sequences cross-referenced with ICTV VMR MSL41 taxonomy, filtered through a multi-stage deduplication and quality-control pipeline to produce a non-redundant set of representative sequences covering 4,200+ viral species across 120+ families.

About This Database

This database integrates NCBI nucleotide records with authoritative ICTV taxonomy to create a curated set of plant virus reference sequences. The pipeline performs:

Host Classification→ ICTV Validation→ Segmented Detection→ Taxonomy Dedup→ Seqkit rmdup→ MMseqs2 0.98→ vclust Clustering

Data sources: NCBI GenBank + ICTV VMR MSL41. Last updated: June 2025.
